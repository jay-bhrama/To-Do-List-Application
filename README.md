# 📝 To-Do List Application

A sleek, minimalistic, and highly functional web application designed to help users organize their daily tasks, increase productivity, and keep track of their goals. Built entirely using vanilla web technologies, this application offers a seamless user experience with real-time state updates.

## ✨ Features

*   **Task Management:** Easily add, mark as completed, and delete tasks.
*   **Persistent Storage:** Uses the browser's `localStorage` API to ensure your tasks are saved even after closing or refreshing the tab.
*   **Interactive UI:** Dynamic task completion styling (striking through completed tasks) and instant DOM updates.
*   **Responsive Design:** Fully optimized layout that works beautifully across mobile devices, tablets, and desktop browsers.

## 🛠️ Tech Stack

*   **Structure:** HTML5
*   **Styling:** CSS3 (Flexbox/Grid layout, modern transitions)
*   **Functionality:** JavaScript (ES6+, DOM Manipulation, LocalStorage API)
